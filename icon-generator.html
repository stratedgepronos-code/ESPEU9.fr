🏀 Générateur d'icônes PWA

Génère toutes les tailles d'icônes nécessaires pour l'appli ESPE U9

🖼️

Clique ici ou glisse ton logo ESPE (PNG carré recommandé, 512x512 min)

💡 Icône maskable : Android peut découper l'icône en rond/carré arrondi. Les versions "maskable" ajoutent un fond vert autour du logo pour éviter qu'il soit coupé.

📋 Mode d'emploi complet

Étape 1 — Générer les icônes

  1. Charge ton logo ESPE ci-dessus (le fichier logo-espe.png de ton site)
  2. Vérifie les aperçus — chaque taille est générée automatiquement
  3. Clique "Télécharger toutes les icônes (ZIP)"
  4. Dézippe le fichier sur ton PC

Étape 2 — Uploader sur Hostinger

  1. Ouvre Hostinger → File Manager
  2. Va dans public_html/images/
  3. Crée un dossier icons (clic droit → New Folder)
  4. Ouvre le dossier icons
  5. Upload tous les fichiers PNG du ZIP dedans

Structure finale :

public_html/
├── manifest.json          ← nouveau
├── sw.js                  ← déjà mis
├── index.html             ← mis à jour
├── images/
│   ├── icons/
│   │   ├── icon-72.png
│   │   ├── icon-96.png
│   │   ├── icon-128.png
│   │   ├── icon-144.png
│   │   ├── icon-152.png
│   │   ├── icon-192.png
│   │   ├── icon-384.png
│   │   ├── icon-512.png
│   │   ├── icon-maskable-192.png
│   │   └── icon-maskable-512.png

Étape 3 — Uploader les autres fichiers

  1. Upload manifest.json dans public_html/ (à la racine, à côté de index.html)
  2. Upload le nouveau index.html dans public_html/

Étape 4 — Tester l'installation

  1. Android : Ouvre espeu9.fr dans Chrome → menu ⋮ → "Installer l'application" ou "Ajouter à l'écran d'accueil"
  2. iPhone : Ouvre espeu9.fr dans Safari → bouton Partager ↗ → "Sur l'écran d'accueil"
  3. L'appli apparaît avec l'icône ESPE sur l'écran d'accueil
  4. En l'ouvrant, elle s'affiche en plein écran sans la barre Chrome !

✅ Vérification (optionnel)

Pour vérifier que tout est bien configuré, ouvre Chrome sur PC, va sur espeu9.fr, puis F12 → onglet Application → Manifest. Tu dois voir toutes les icônes et les infos de l'appli sans erreur.
