Guide parent

Découvrir les fonctionnalités du site ESPE Basket U9

Saison 2025-2026

2 / 11

Écran d'entrée (connexion)

Première visite : espeu9.fr → vous arrivez sur la page « Accès membres »

  • Connexion : identifiant + mot de passe, option « Se souvenir de moi »
  • Création de compte : identifiant, mot de passe, email (optionnel), choix du joueur (votre enfant) et du lien de parenté (Papa / Maman)
  • Mot de passe oublié : envoi d’un lien de réinitialisation par email
Une fois connecté, vous accédez à l’ensemble du site (accueil, matchs, convocations, tchat, compte).
3 / 11

Accueil

Tableau de bord après connexion

  • Actualités : infos du club, annonces, résultats récents
  • Anniversaires : prochains anniversaires des joueurs de l’équipe
  • Accès rapide : liens vers Matchs, Convocations, Tchat, Compte
En cliquant sur un joueur (anniversaire ou liste), vous pouvez voir son détail si disponible.
4 / 11

Matchs

Calendrier et détails des rencontres

  • Liste des matchs : date, heure, adversaire, lieu
  • Détail d’un match : gymnase, adresse, équipe adverse
  • Lien Google Maps : en cliquant sur le nom du gymnase, une carte s’ouvre pour vous guider
Plus besoin de copier-coller l’adresse : un clic sur le gymnase ouvre directement l’itinéraire.
5 / 11

Convocations

Répondre pour chaque match

  • Voir les convocations : matchs pour lesquels votre enfant est convoqué
  • Répondre : indiquer « Présent » ou « Absent » pour chaque convocation
  • Rappel automatique : la veille du match, une notification push vous rappelle l’événement (si les notifications sont activées)
Pensez à activer les notifications dans « Compte » pour ne rien manquer.
6 / 11

Indisponibilités (absences)

Signaler quand votre enfant ne peut pas être présent

  • Où ? Dans « Compte » (section parent)
  • Comment ? Ouvrir le calendrier, cliquer sur le premier jour d’indisponibilité, puis sur le dernier jour
  • Effacer : bouton pour réinitialiser la plage et recommencer
Le coach voit ces plages et peut mieux organiser les convocations.
7 / 11

Tchat / Messagerie

Échanges avec le club et les autres parents

  • Fil de discussion : messages avec texte, images, liens, musique (Spotify, YouTube…)
  • Vos messages : vous pouvez supprimer uniquement vos propres messages
  • Accès : depuis le menu « Tchat » ou le lien Messagerie sur la page d’accueil
Un badge indique les messages non lus. Idéal pour les infos de dernière minute.
8 / 11

Compte (profil)

Gérer votre profil et vos préférences

  • Profil : identifiant, joueur lié (votre enfant), lien de parenté
  • Indisponibilités : accès au calendrier des absences (voir slide précédente)
  • Notifications push : activer ou désactiver les rappels (convocations, rappels matchs, etc.)
Recommandé : activer les notifications pour recevoir les convocations et le rappel la veille des matchs.
9 / 11

Notifications push

Rester informé même sans ouvrir le site

  • Convocations : quand le coach convoque votre enfant pour un match
  • Rappel match : la veille du match, rappel automatique (date, lieu, heure)
  • Activation : dans « Compte » → section Notifications, accepter quand le navigateur le demande
Sur téléphone, autorisez les notifications pour le site dans les paramètres du navigateur si besoin.
10 / 11

Déconnexion

Quitter l’espace membres en toute sécurité

  • Où ? Bouton « Déco » en haut à droite (ou « Déconnexion » dans le menu mobile)
  • Effet : vous êtes redirigé vers l’écran d’entrée (page de connexion)
  • Reconnexion : identifiant + mot de passe sur gate.html
Après déconnexion, impossible d’accéder au contenu réservé sans se reconnecter.
11 / 11

Récapitulatif

En résumé pour les parents

  • Connexion / inscription sur l’écran d’entrée
  • Accueil : actualités et anniversaires
  • Matchs : calendrier + lien Google Maps (gymnase)
  • Convocations : répondre présent/absent + rappel la veille
  • Indisponibilités : calendrier (plage de dates) dans Compte
  • Tchat : messagerie commune du groupe
  • Compte : profil, absences, notifications push
  • Déconnexion : retour à l’écran d’entrée

Merci et à bientôt sur le terrain !
